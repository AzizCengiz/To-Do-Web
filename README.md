# 📝 To-Do Web Application

A simple and user-friendly task management application developed using HTML, CSS, and JavaScript. This project allows users to create, edit, and delete tasks while providing a clean and responsive user interface.

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* User-friendly interface
* Responsive design
* Dynamic interactions with JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Git
* GitHub

## 📂 Installation

1. Clone the repository:

```bash
git clone https://github.com/AzizCengiz/To-Do-Web.git
```

2. Navigate to the project directory:

```bash
cd To-Do-Web
```

3. Open the `index.html` file in your browser.
